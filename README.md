# 🎄 圣诞3D交互式网页

一个精美的圣诞主题3D交互式网页应用，使用Three.js和MediaPipe手势识别技术打造。

## ✨ 功能特性

### 🎨 视觉效果
- **3D圣诞树**: 由1500+粒子组成的动态圣诞树
- **三种视图模式**:
  - 🌲 树形模式 - 粒子螺旋排列成圣诞树
  - 💫 散开模式 - 粒子3D球形分布
  - 🔍 聚焦模式 - 放大查看单张照片
- **精美装饰**: 金色球、绿色方块、红色球、糖果手杖
- **礼物盒**: 5个彩色礼物盒围绕树底
- **泛光效果**: UnrealBloom后期处理
- **动态雾效**: 营造梦幻氛围

### 🎵 音频体验
- **背景音乐**: 循环播放圣诞歌曲
- **音量控制**: 默认40%音量
- **快捷键**: 按 `M` 键切换播放/暂停
- **控制按钮**: 左下角金色圆形按钮

### 💬 圣诞祝福
- **10条祝福语**: 自动轮播显示
- **切换间隔**: 每5秒切换一次
- **淡入淡出**: 平滑过渡动画

### 📸 照片上传
- **添加回忆**: 上传个人照片到场景
- **金色相框**: 自动添加优雅相框
- **多张支持**: 可上传多张照片
- **3D展示**: 照片参与场景互动

### 👋 手势控制
- **捏合**: 食指+拇指 → 聚焦照片
- **握拳**: 手掌收紧 → 返回树形
- **张开**: 手掌展开 → 散开模式
- **移动**: 手掌移动 → 控制旋转

## 🚀 使用方法

### 方式一: 直接打开（推荐）
1. 确保 `Christmas.mp3` 音乐文件与 `圣诞.html` 在同一目录
2. 双击打开 `圣诞.html` 文件
3. 允许浏览器访问摄像头（可选，用于手势控制）
4. 点击页面任意位置开始播放音乐

### 方式二: 本地服务器
```bash
# 使用Python
python -m http.server 8000

# 使用Node.js
npx serve

# 然后访问 http://localhost:8000/圣诞.html
```

## ⌨️ 快捷键

- **M** - 播放/暂停背景音乐

## 🎁 自定义选项

### 更换背景音乐
打开 `圣诞.html`，找到第176行：
```html
<source src="Christmas.mp3" type="audio/mpeg">
```
替换为你的音乐文件路径。

### 修改祝福语
在第253-264行找到 `blessings` 数组，修改或添加祝福语。

### 调整粒子数量
在第164-168行修改 `CONFIG.particles`:
```javascript
particles: {
    count: 1500,      // 主要粒子数量
    dustCount: 2500,  // 灰尘粒子数量
}
```
